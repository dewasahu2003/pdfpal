steps to make it work on your machine
- start docker
- `docker compose up`
- `./tiny-windows.bat`
- set up virtual env
- `python -m venv venv`
- `venv/Scripts/activate`
- `pip install -r reqtiny.txt`
- `streamlit run tiny_rag.py`
